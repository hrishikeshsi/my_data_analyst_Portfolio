# Data Analyst Portfolio

This is the personal portfolio website of **Hrishikesh Raghuvanshi**, a Data Analyst specializing in Python, SQL, and Visualization.

## Features
- **Dark Mode UI**: Pure black aesthetic with neon accents.
- **Interactive Animations**: GSAP scroll reveals and custom cursor.
- **Data Analyst Assistant**: An animated CSS-only character that types code and analyzes data.
- **Live Background**: A generated canvas animation simulating real-time data fluctuations.

## Technologies
- HTML5
- CSS3 (Variables, Keyframes, Flexbox/Grid)
- JavaScript (Vanilla, Canvas API)
- GSAP (GreenSock Animation Platform)

## Setup
No build process required. Just open `index.html` in your browser.

## Deployment
Ready for deployment on GitHub Pages or Netlify.
